# Security Policy

## Scope

This is a personal static website with no backend, no authentication, and no user data. The attack surface is limited to the frontend code and the CI/CD pipeline.

## Reporting a Vulnerability

If you find a security issue in this repository (e.g. a misconfigured workflow, exposed secret, or vulnerable dependency), please report it privately via GitHub's [private vulnerability reporting](https://github.com/serenacofano/serenacofano.github.io/security/advisories/new) rather than opening a public issue.

I will respond as soon as possible and address valid findings promptly.

## Supported Versions

Only the current `master` branch is maintained.
